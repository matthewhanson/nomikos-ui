# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2025-12-14

### Added
- Fantasy-themed UI with purple/gold gradients and sword motifs
- Three interaction modes: Chat, Answer, and Search
- 100+ curated example queries for exploring the knowledge base
- Markdown rendering for rich formatted responses
- Responsive design for mobile and desktop
- GitHub Pages deployment workflow
- Configurable API endpoint via `VITE_NOMIKOS_URL` environment variable
